Machine Learning Portfolio
🎓 Overview

This repository showcases a collection of machine learning projects developed as part of my self-learning journey in Data Science and Artificial Intelligence.
It demonstrates practical experience in data preprocessing, exploratory data analysis, feature engineering, model building, hyperparameter tuning, and evaluation across different types of machine learning problems.

📁 Projects
🌍 Country Clustering (Unsupervised Learning)
* Applied K-Means clustering to group countries based on socio-economic indicators
* Used PCA for dimensionality reduction
* Evaluated clustering performance using multiple metrics (Elbow Method, Silhouette Score, etc.)
* Selected 4 meaningful clusters for better interpretation of global patterns

📧 Email Spam Detection (NLP Classification)
* Built multiple text classification models to detect spam emails
* Applied TF-IDF vectorization for feature extraction
* Compared several models including SVM, Logistic Regression, Random Forest, Naive Bayes, and Gradient Boosting
* Best performing model: Support Vector Machine (SVM)

🏡 House Price Prediction (Regression)
* Predicted house prices using features such as location, bedrooms, and bathrooms
* Performed data cleaning, outlier removal, and exploratory data analysis
* Trained multiple regression models: Linear Regression, Lasso, and Decision Tree
* Used GridSearchCV for hyperparameter tuning
* Best model achieved 85% accuracy
  
🛠️ Tools & Technologies
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, NLP (TF-IDF), PCA, K-Means, GridSearchCV

The goal of this portfolio is to demonstrate my ability to build end-to-end machine learning solutions and prepare for advanced studies in Data Science and Artificial Intelligence.
