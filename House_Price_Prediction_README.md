📌 Project Objective

The goal of this project is to build a machine learning model capable of predicting house prices based on key features such as:

Location
Number of bedrooms (BHK)
Number of bathrooms
🧹 Data Preprocessing

Before training the models, the dataset was carefully cleaned:

Removal of outliers to improve data quality
Handling inconsistent or unrealistic values
Feature cleaning and transformation for better model performance
📊 Exploratory Data Analysis (EDA)

Data visualization was performed to understand patterns in the dataset:

Distribution of house prices
Relationship between features and price
Impact of location on pricing
Detection of trends and anomalies
🤖 Machine Learning Models

Several regression models were trained and compared:

Linear Regression
Lasso Regression
Decision Tree Regressor
🔧 Model Optimization

To improve performance, GridSearchCV was used for hyperparameter tuning:

Tested multiple parameter combinations
Applied cross-validation for reliable evaluation
Selected the best configuration for each model
🏆 Results
The best model achieved an accuracy of 85%
Linear Regression with optimized parameters performed best on this dataset
